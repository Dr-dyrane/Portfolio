## Getting Started

To explore or contribute to this portfolio, follow the steps below:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Dr-dyrane/Portfolio.git
   ```

2. **Installation:**
   ```bash
   cd Portfolio
   npm install
   ```

3. **Run Locally:**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
